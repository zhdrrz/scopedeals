How this self-running site works

  1. Products live in data/products.json — this is the single source of truth
  2. GitHub Actions runs scripts/update_prices.py every Monday & Thursday to scrape latest prices
  3. Then scripts/build_site.py generates public/index.html from the template + data
  4. The updated site auto-deploys to Netlify — zero manual work needed
This admin panel lets you: add/edit/delete products, manage affiliate links, and export the updated products.json to commit back to GitHub.

To update affiliate links: Go to the Affiliate Links tab below, paste your affiliate URLs for each product, and click Export JSON to download the updated file. Then replace data/products.json in your GitHub repo.
